**AETERNA** is a high-end, immersive luxury watch experience built with precision and passion. A showcase of modern web technologies, it features a procedural 3D watch customizer, interactive anatomy, and a seamless cinematic user experience.

[🚀 LIVE DEMO](https://zihad2003.github.io/watchcom-/)

---

## ✨ Features

- **🛡️ 3D Immersive Hero**: A procedurally rendered 3D watch using **Three.js** that reacts to your every mouse movement.
- **🛠️ Bespoke Customizer**: Real-time watch configuration with live material and pricing updates.
- **🌌 Cinematic GSAP Animations**: High-end scroll-triggered reveals and transitions.
- **🦴 Exploded View Anatomy**: Interactive 3D CSS decomposition of the watch components.
- **📜 Heritage Timeline**: A journey through the centennial history of AETERNA.
- **📱 Responsive Luxury**: Fully optimized for mobile, tablet, and desktop with a focus on premium UX.

---

## 🛠️ Tech Stack

- **Graphics**: [Three.js](https://threejs.org/) (Procedural 3D Engine)
- **Animation**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, MotionPath)
- **Frontend**: Vanilla HTML5, CSS3 (Modern Flex/Grid), JavaScript (ES6+)
- **Typography**: Google Fonts (Cormorant Garamond, Montserrat)

## 🚀 Installation & Local Development

No heavy installation or framework is required. Just pick up your favorite browser and go.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/zihad2003/watchcom-.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd watchcom-
   ```
3. **Open index.html**:
   - Double-click `index.html` file OR
   - Run a local server: `npx serve .`
