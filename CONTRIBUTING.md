# Contributing to AETERNA

First off, thanks for taking the time to contribute! ❤️

As a luxury brand project, we value precision, design, and performance above all else. Your contributions help us make **AETERNA** better for the whole community.

---

### How Can I Contribute?

#### 🛠️ Improvement Suggestions
-   Refine the **Three.js** watch geometry or materials.
-   Improve the **GSAP** scroll performance.
-   Fix **Typographical** or **Design** inconsistencies.
-   Add **New Sections** (e.g., Sustainability, Retailer Locator, etc.).

#### 🐞 Reporting Bugs
-   Open an issue using the **Bug Report** template.
-   Provide clear steps to reproduce the bug.
-   Mention your browser and screen resolution.

#### 🏗️ Pull Requests
1.  **Fork** the repo.
2.  Create your feature branch (`git checkout -b feature/AmazingFeature`).
3.  Commit your changes following the **commit naming convention** (`feat:`, `fix:`, `chore:`, `docs:`).
4.  Push to the branch (`git push origin feature/AmazingFeature`).
5.  Open a **Pull Request** and describe your changes.

---

### Code Style Guidelines

-   Keep **index.html** organized to maintain its modular feel.
-   Follow the existing **CSS variables** for colors and fonts.
-   Ensure **responsiveness** on all viewports.
-   Comment your **Javascript** logic for readability.

### Achievement Hunters

Want to help us reach the **Starstruck** or **Galaxy Brain** achievements? 
-   Star the repo if you find it inspiring!
-   Engage in the **Discussions** tab with questions or suggestions.
